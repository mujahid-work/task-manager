# 🧰 Task Management System (Phase 1 - Modern PHP Only)

At this stage of project, the foundation of a full-stack Task Management System built **without any frameworks like Laravel**, using **modern PHP 8.x**, clean architecture, and best development practices.

---

## 🎯 Phase 1 Objective

Build a modular backend for managing users and tasks using:

- ✅ PHP 8.x features
- ✅ OOP with SOLID principles
- ✅ PSR standards (PSR-1, 4, 7, 12)
- ✅ Composer (autoloading, package management)
- ✅ MySQL (via PDO) for persistent task/user storage

---

## 📚 Concepts & Architecture

- **Entities**: `User`, `Task` – defined using clean OOP principles  
- **Repositories**: Interface-based data access using PDO  
- **Services**: Business logic layer (e.g., task assignment)  
- **Controllers**: Handle routing and request processing  
- **Routing**: Simple front controller with manual routing logic  
- **HTTP Layer**: PSR-7-style Request/Response objects
